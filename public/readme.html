Channel Feed: a minimalistic YouTube channel curator

Channel Feed is a lightweight dashboard that shows content only from the channels you care about. No recommendations, no distractions, just the latest uploads and posts you asked for.

Focused feed

Only channels you follow. Organized by channel so it stays easy to scan.

Smart filters

Pick long videos, Shorts, and community posts per channel.

Local and private

Your API key and settings stay in your browser storage.

Purpose

The goal is clarity: track creators you like without YouTube recommendations pulling you elsewhere.

  • Collect content from hand-picked channels only.
  • Keep a clean view of new uploads over a chosen time range.
  • Stay in control of what counts as a video (long, Shorts, or community).

How to use Channel Feed

1

Open Settings and add your API key

Click Settings in the header, paste your YouTube Data API key, and click Apply.

2

Search for channels

Use the sidebar search box and click the plus button to follow a channel.

3

Choose content types

Use the channel settings icon to include long videos, Shorts, or community posts.

4

Refresh when you want a full update

Refresh pulls the latest content across all channels, using your current filters.

Tip: When you add a new channel, its items are pulled immediately and merged into the feed.

Step-by-step: get a YouTube Data API key

1

Open Google Cloud Console

Go to console.cloud.google.com and sign in with your Google account.

2

Create or select a project

Click the project selector at the top left and choose an existing project or create a new one.

3

Enable the YouTube Data API v3

Go to APIs & Services > Library, search for "YouTube Data API v3", and click Enable.

4

Create credentials

Open APIs & Services > Credentials, click Create Credentials, and choose API key.

5

Restrict the key (recommended)

Click the key to edit it, then set an HTTP referrer restriction for your domain or localhost.

6

Copy the API key into Channel Feed

Paste the key in Settings and click Apply. You are ready to search and curate.

If search results do not appear, confirm the API is enabled and the key restrictions allow your site.

Notes on API usage

YouTube Data API v3 has a daily quota. Searches cost more than refreshes.

With the default 10,000 units/day: about 100 searches per day (100 units each). With 10 channels and long/shorts only, a refresh is about 20 units, so roughly 500 refreshes per day.

  • Searching for channels: higher cost per query.
  • Refreshing feed: optimized playlist and batch requests.
  • Community posts: separate request per channel.
